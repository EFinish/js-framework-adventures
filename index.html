Hello.

The purpose of this application is to serve as a collection of code for the same thing written in different ways. Each 'thing' will be

  • An HTML webpage
Each of these webpages will
  • Have a large header title featuring the technology used and the word example. i.e "Angular6 Example"
  • Have a form for submitting information to a url Each form will
    • Have an input for the user's name (type text)
    • Have an input for the user's age (type number)
    • Have an input for the user's birthday (whatever works)
    • Have an input for the user's gender (type radio, options male, female, other)
      • If the user selects other for gender, show a text box beneath prompting for the user to type in their gender association
    • Have an input for the user's opinion on whether they are sure they having a nice day (type select, 3 options yes, no, maybe so)
    • Have a button for the user to submit the button to a url that will return a 200 http response.
    • Have a button for the user to submit the button to a url that will return a 500 http response.
    • Upon click the button will
      • Validate that all of the data in the form field exists and that they are all of a valid input format.
      • Validate that the age and birthday given are valid to coexist truthfully given the current time.
      • Let the user know if any of the data is invalid and why.
    • the form with all of the data must be sumitted it's corresponding url and let the user know if the request succeeded accordingly.
  • More requirments coming...
